方法的动态参数列表

基于动态参数列表的数据接收方式(形如String...names)

Mar 01, 2021

动态参数(可变参数)

动态参数在方法内部,可以当作数组处理
在调用方法的时候,可以按如下三种方式进行调用:

  1. 接收数组
  2. 接收一个数据
  3. 接收n多个数据

public class ParameterTest 
{
    public static void main(String[] args)
    {
        String pnames[]={"Steve","Amy","Alex"};
        ParamaterTest.Select(pnames);
        //方式1
        ParamaterTest.Select("Steve");
        //方式2
        ParamaterTest.Select("Steve","Amy","Alex");
        //方式3
    }

    public static void Select(String...pnames)
	{
		for(String name:pnames)
		{
			System.out.println("Select "+name);
		}
	}
}

拓展:for-each循环数组

基本语句:

1
2
3
4

for ( 变量类型  变量名 : 数组名 ) 
{  
    需要执行的循环语句;
}

其中变量类型与数组内元素类型相同

for-each循环会从下标为'0’的元素开始遍历,直到最后一个元素,并逐次将对应元素变量的值赋给前面设置的变量名里
该方法相对于一般的for循环更加方便

1

String pnames[]={"Steve","Amy","Alex"} ;
